You must have both [nodeJS](https://nodejs.org/en/download/package-manager/) and [yarn](https://yarnpkg.com/lang/en/docs/install) or [npm](https://www.npmjs.com/get-npm) installed.

To install dependencies:

```
    $ yarn install
```
or 
```
    $ npm install
```

To run tests:

```
    $ yarn test
```
```
    $ npm run test
```
